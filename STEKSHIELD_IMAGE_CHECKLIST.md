## Image Format Notes
- Supported formats: PNG, JPG, JPEG
- Recommended format: PNG for better quality
- High resolution recommended for best display quality
- Images will automatically scale to fit the layout

## After Upload
Once you've uploaded all images to the folder:
`src/images/projects/stekshield-crm/`

`http://localhost:3000/projects/stekshield-crm`

If any images are missing, the system will automatically display a gray placeholder with "Placeholder image" text, so you can see which images still need to be added.
